# Stage 4: Scene Planning

Break a chapter into 3-5 writeable scenes.

## Requirements

**Length:** 60–80 words per scene description

**Input:** Chapter number from outline

## For Each Scene

Include:
- **Location:** Where does this scene take place?
- **Characters:** Who is present?
- **Conflict:** What tension drives this scene?
- **Purpose:** How does this scene advance the story?
- **MUST:** What MUST this scene establish/do?
- **MUST NOT:** What MUST NOT happen yet?

## CRITICAL: Continuity Anchor

Include at the top:

```
## Previous Chapter (X-1) Recap
[2 sentences: How did the previous chapter end? What was established?]
```

## Format

```
# Chapter X Scenes

## Previous Chapter Recap
[2 sentences]

## Scene 1: [Name]
**Location:** [Description]
**Characters:** [Who is present]
**Conflict:** [What tension drives this]
**Purpose:** [How it advances plot]
**MUST:** [What this scene must do]
**MUST NOT:** [What cannot happen yet]

## Scene 2: [Name]
[Same structure]
...
```

## Constraints

- 60-80 words per scene description
- 3-5 scenes per chapter (no more, no less)
- Each scene should end with a reason to turn the page
- Don't front-load resolutions

## Memory Anchor

Before writing, read:
- `premise.md`
- `story_spine.md`
- `chapter_outline.md`
- `scenes/chapter_(X-1)_scenes.md` (for continuity)

## Output

Save as `scenes/chapter_XX_scenes.md` (e.g., `chapter_01_scenes.md`)
