# Stage 3: Chapter Outline (50 Chapters)

Create a 50-chapter outline following the story spine.

## Requirements

**Length:** 80–120 words per chapter description (STRICT)

**Structure:**
- 50 chapters total
- Chapters 1-24: First act through midpoint
- Chapter 25: Midpoint twist
- Chapters 26-44: Second act
- Chapter 45: Final act begins
- Chapters 45-50: Climax and resolution

## Rules

1. Each chapter MUST advance the plot
2. Do NOT summarize the entire story in one chapter
3. Chapters must build sequentially toward midpoint and climax
4. Chapter 25 contains the midpoint twist
5. Chapter 45 begins the final act
6. No chapter is filler

## Format

```
# Chapter Outline

## Chapter 1: [Title]
[80-120 words describing what happens]

## Chapter 2: [Title]
[80-120 words]
...
```

## Constraints

- 80-120 words EXACTLY per chapter
- If a chapter description runs long, it means you're summarizing too much — cut
- Each chapter should have a clear purpose and ending point
- Chapter titles are brief (3-6 words)

## Memory Anchor

Before writing, read:
- `premise.md`
- `story_spine.md`

## Output

Save as `chapter_outline.md` in project root.
