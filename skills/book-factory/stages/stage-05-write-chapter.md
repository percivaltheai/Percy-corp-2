# Stage 5: Write Chapter

Write the prose draft for a single chapter.

## Requirements

**Length:** 2,500–3,500 words (FIRM ceiling)

## Memory Anchor (READ FIRST)

Before writing, read these files:
```
- premise.md
- story_spine.md
- chapter_outline.md
- scenes/chapter_XX_scenes.md
- chapters/chapter_(X-1).md (previous chapter for continuity)
```

## Rules

1. Follow the scene outline exactly
2. Maintain the narrative voice from the premise
3. Do NOT resolve major plot arcs early
4. Stay within word count (ceiling: 3,500)
5. End each scene with a reason to continue
6. Include sensory details, dialogue, emotional beats

## If Exceeding Word Count

If you reach 3,500 words and the chapter isn't done:
- STOP
- Note where you cut off
- Outline the next chapter first
- Split into two chapters

## Format

```
# Chapter X: [Title]

## Scene 1
[Prose...]

## Scene 2
[Prose...]
...
```

## Constraints

- First-person or third-person (consistent with premise)
- No meta-commentary within prose
- Literary prose, not AI-slop
- Pacing: vary sentence length
- No info-dumping — weave backstory into action

## Output

Save as `chapters/chapter_XX.md` (e.g., `chapter_01.md`)
