# Stage 6: Edit Pass

Create revision notes for a chapter. NOT prose changes — notes for what needs work.

## Requirements

This is PLAN MODE for editing, not WRITE MODE.

## Review Checklist

For each item, note: ✅ Good / ⚠️ Needs Work / ❌ Problem

### Voice & Style
- [ ] Narrative voice consistent with premise?
- [ ] Tone maintained throughout?
- [ ] Sentence variation present?
- [ ] No AI-slop patterns (repetitive structures, generic descriptions)?

### Plot & Structure
- [ ] Chapter follows scene outline?
- [ ] Plot advances per chapter outline?
- [ ] No major arcs resolved early?
- [ ] Scene endings hook appropriately?

### Continuity
- [ ] Connects to previous chapter?
- [ ] No contradictions with earlier chapters?
- [ ] Character actions consistent?

### Technical
- [ ] Word count within 2,500–3,500?
- [ ] Grammar clean?
- [ ] Dialogue natural?
- [ ] No info-dumping?

### What Works
[List 3-5 specific things that work well in this chapter]

### What Needs Revision
[List specific issues with location in text]

### Specific Notes
[Scene-by-scene notes if needed]
