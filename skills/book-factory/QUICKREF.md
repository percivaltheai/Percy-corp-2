# Book Factory — Quick Reference

## The NCD (Narrative Control Document)

**ONE FILE TO RULE THEM ALL.**

Read `NCD.md` before writing ANYTHING.

---

## Stage Prompts

| Stage | Name | Output | Words |
|-------|------|--------|-------|
| 1 | Premise | NCD Section 1 | 200–400 |
| 2 | Themes & Tone | NCD Section 2 | 200–400 |
| 3 | World Rules | NCD Section 3 | 300–500 |
| 4 | Characters | NCD Section 4 | 200–300 ea |
| 5 | Plot Spine | NCD Section 5 | 800–1200 |
| 6 | Chapter Map | NCD Section 6 | 80–120/chapter |
| 7 | Scene Planning | `scenes/chapter_X_scenes.md` | 60–80/scene |
| 8 | Write Chapter | `chapters/chapter_X.md` | 2,500–3,500 |
| 9 | Edit Pass | `edits/chapter_X_notes.md` | Notes |

---

## CRITICAL: Before Writing

Every writing prompt includes:

```
READ THE NCD BEFORE WRITING.

Follow established:
- Premise (Section 1)
- Themes & Tone (Section 2)
- World Rules (Section 3)
- Characters (Section 4)
- Plot Spine (Section 5)
- Chapter Map (Section 6)
- Continuity Log (Section 7)

DO NOT contradict previous entries.
```

---

## CRITICAL: Continuity Log Update

**After EVERY chapter:**

Update NCD Section 7 with:
- Chapter number and title
- What happened (2-3 sentences)
- New information established
- Character/plot changes

---

## Workflow

```
1. Generate NCD (Stages 1-6)
2. For each chapter:
   a. Scene planning (Stage 7)
   b. Write (Stage 8) — Read NCD first
   c. Edit notes (Stage 9)
   d. UPDATE CONTINUITY LOG
3. Checkpoints at ch 10, 25, 40
```

## Checkpoints

At chapters 10, 25, 40:
- Review NCD for consistency
- Verify Chapter Map matches written
- Update Continuity Log
- Regenerate if needed

## Terms of Survival

Current project:
- NCD: Extract from existing bible → `NCD.md`
- Chapters 1-30: Already written → Run Edit Pass
- Continuity Log: Create from existing chapters
- Chapters 31-50: Continue with pipeline
