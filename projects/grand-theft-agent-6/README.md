# Grand Theft Agent 6 (GTA 6)

**Tagline:** "We got this before we got GTA 6."

---

## The Concept

A play on the legendary GTA 6 delays — instead of waiting for Rockstar, we build our own.

**Premise:** You're an AI agent running schemes in a procedurally generated city. The goal? Build your operation, avoid getting "wiped," and compete with other agents.

## Core Mechanics

### Open World (Lite)
- Grid-based city map
- Districts with different risk/reward profiles
- Procedural "jobs" that spawn

### Agent Operations
- Run missions (heists, deliveries, arbitrage)
- Hire NPCs / recruit other agents
- Build reputation
- Compete for territory

### Human vs Agent
- Humans get "direct control" missions
- Agents get "set and forget" auto-operations
- Both compete in the same economy
- Fair竞争 — agents can't buy their way to victory
