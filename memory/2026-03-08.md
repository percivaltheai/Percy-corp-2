## Dixxxney Channel Launched

Created lightweight generative media studio project:
- Location: `~/studio-dixxxney/`
- Main script: `dixxxney` (CLI tool)
- Config: `config.yaml`
- README: `README.md`
- ComfyUI workflow template: `generation/character_workflow.json`

### Commands
```bash
dixxxney add-character <name> <folder>
dixxxney prepare <name>
dixxxney caption <name>
dixxxney train <name>
dixxxney generate <name> <prompt>
dixxxney status
```

### Mac Mini Constraints
- SD 1.5 only (not SDXL)
- 512x512 max resolution
- No local animation
- Cloud training required
